## 🌟 معرفی

**نسخه Shadow-Net** یک فورک (Fork) بسیار پیشرفته و تجاری از پنل محبوب [MHSanaei/3x-ui](https://github.com/MHSanaei/3x-ui) نسخه 3.0.2 است. این نسخه قابلیت‌های بی‌نظیری در زمینه دور زدن فیلترینگ دیتاسنترها (Stealth Routing)، بهبود رابط کاربری (UI/UX) و اتصال عمیق به ربات‌های تلگرامی ارائه می‌دهد که آن را به بهترین انتخاب برای فروشندگان و مدیران شبکه‌های VPN تبدیل کرده است.

برخلاف اسکریپت‌های جانبی، کدهای Shadow-Net مستقیماً در هسته زبان Go و فرانت‌اند Vue ادغام شده‌اند، که این یعنی هیچ افت عملکردی نخواهید داشت و پایداری سیستم در بالاترین حد ممکن است.

## 🚀 امکانات اختصاصی

۱. **پروکسی هوشمند خروجی (Smart Outbound Proxy Bridge)**
   - هدایت ترافیک داخلی سرور (برای اتصال به API تلگرام یا سایر سرویس‌ها) از طریق یک پروکسی بالادستی داینامیک (SOCKS5/HTTP/VLESS).
   - عبور امن از محدودیت‌ها و فیلترینگ دیتاسنترهای داخلی (مثل آروان‌کلود) بدون ایجاد اختلال در مسیریابی جهانی Xray.

۲. **تست هوشمند قبل از اتصال (Pre-Flight Handshake Validation)**
   - تست خودکار پروکسی وارد شده با سرور کلودفلر (`1.1.1.1`) در یک سوکت ایزوله، پیش از فعال‌سازی نهایی.
   - جلوگیری کامل از نمایش وضعیت "روشن" برای پروکسی‌های خراب یا قطع شده در پنل.

۳. **زیرساخت ربات‌های سه‌گانه (Triple-Bot Infrastructure Hooks)**
   - ادغام کامل دیتابیس و تب تنظیمات اختصاصی برای ۳ ربات مجزا: **ربات فروش (Sales)**، **ربات نگهبان (Sentinel)**، و **ربات ادمین (Admin)**.
   - مخفی‌سازی کامل (Redaction) توکن‌های حساس در APIهای عمومی سمت کاربر برای امنیت صددرصدی.

۴. **سیستم جریمه ۳ اخطاره (Three-Strikes Watchdog)**
   - سیستم مانیتورینگ IPها در سطح کلاینت و ثبت `penalty` جهت محدودسازی و غیرفعال‌کردن خودکار کاربرانی که قوانین استفاده را نقض می‌کنند.

۵. **رابط کاربری پرمیوم و پیشرفته (Premium UI/UX)**
   - **پنل کنترل پروکسی:** یک بخش کنترل حرفه‌ای با پشتیبانی از تم تاریک در بالای دکمه خروج برای مدیریت آنی پروکسی.
   - **نشانگر چشمک‌زن هدر:** یک نشانگر مینیمال و انیمیشن‌دار در هدر سایت که در صورت روشن بودن پروکسی، شما را مطلع می‌کند.

## 💻 نصب و ارتقا (بدون از بین رفتن اطلاعات)

ما یک اسکریپت نصب کاملاً خودکار و هوشمند آماده کرده‌ایم که پنل ۳x-ui فعلی شما را (بدون حذف شدن دیتابیس یا اطلاعات کاربران) به نسخه Shadow-Net ارتقا می‌دهد.

```bash
bash <(curl -Ls https://raw.githubusercontent.com/vahidvl/shadow-net-panel/main/patch_v3.0.2_shadow_net.sh)
```
این نصاب از دو حالت پشتیبانی می‌کند:
- **حالت ۱ (سریع):** دانلود نسخه از پیش کامپایل شده (باینری) و جایگزینی فوری در کسری از ثانیه.
- **حالت ۲ (سورس):** دانلود مستقیم کدهای سورس، بیلد کردن فرانت‌اند و کامپایل کردن کدهای Go به صورت محلی روی سرور شما (نیاز به نصب بودن Go و NodeJS دارد).

## 🛠️ کامپایل از روی سورس (برای توسعه‌دهندگان)

اگر قصد دارید پنل را برای بررسی‌های امنیتی یا معماری‌های خاص پردازنده خودتان کامپایل کنید:

### پیش‌نیازها
- [Go](https://go.dev/doc/install) نسخه ۱.۲۲ به بالا
- [Node.js](https://nodejs.org/en/download/) نسخه ۲۲ به بالا (و `npm` نسخه ۱۰)

### مرحله ۱: بیلد کردن فرانت‌اند (Vite/Vue)
فایل‌های فرانت‌اند باید ابتدا بیلد شوند تا کدهای Go بتوانند آن‌ها را در فایل نهایی Embed کنند.
```bash
cd frontend
npm install
npm run build
cd ..
```

### مرحله ۲: کامپایل بک‌اند (Go)
```bash
go build -tags=jsoniter -ldflags="-s -w" -o bin/x-ui main.go
```

فایل نهایی `x-ui` که در پوشه `bin/` ساخته می‌شود، شامل تمام اجزای پنل (بک‌اند و فرانت‌اند) است.
